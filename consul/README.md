consul
=========

Установка кластера Consul

Требования
------------

- **Almalinux**: 8

Переменные
--------------

- *datacenter*: Имя Дата центра.

Название группы в инвентаре для кластера: cluster.

Пример плэйбука
----------------

    - hosts: servers
      roles:
         - { role: consul, datacenter: 21 }
